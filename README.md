# [TouHou MIX] 人声歌曲曲包合集

## 📖 项目简介
这是一个非官方的曲包合集仓库，收集了 [TouHou Mix] 的所有含人声音频的曲目资源，方便玩家下载与备份。  

**注意**：本项目仅用于学习、交流与备份目的，请勿用于商业用途。所有音乐版权归原作者所有。

## ⚡ 快速开始
### 下载方式
1. **使用Git**（推荐）：  
   ```bash
   git clone https://github.com/YangQwQ/THMIX-Vocal-Songs.git
   ```
2. **Onedrive**：
   - 如果你现在正在用Onedrive访问这个文件，那么你可以直接点击文件下载
   - 也可以将以时间段内的音频汇总的文件夹打包下载

### 使用方法
1. 将文件放置到游戏文件目录之中：
   - **Android**：`「内部储存」/Android/data/kailang.touhoumix/files`
   - **PC**：暂无此功能
   - **IOS**：暂无此功能
2. 游戏内需要下载对应歌曲或导入该歌曲的Midi：
   - 打开游戏，在歌曲商店搜索该文件的文件夹编号或者文件名
      + 比如你下载的是 `996 魔法少女小圆/02 Connect/01 【Connect】/1eb0a272d8ed5489507ddd86fa7757f5.ogg`，那么可以在游戏内搜索 `996` 或者 `1eb0a272d8ed5489507ddd86fa7757f5` 来找到该歌曲
   - 如果你是自行导入的midi文件，那么文件和音频文件命名一致即可（不含后缀名）
